\# Option\_Drone



The primary objective of this student-led project is the collaborative design and construction of a fully functional drone. 



To facilitate teamwork and project management, the cohort is divided into several dedicated teams, each comprising 2 to 4 members. Within this repository, every folder serves as the distinct workspace for a specific team.



This project is a direct continuation of the foundational work established by the previous year's students. You can review their progress, access their codebase, and leverage their contributions by visiting last year's repository:

\[EnseaDronesOption - Previous Year's GitHub](https://github.com/Lynxlegrand/EnseaDronesOption)
